# Digital-Clock-with-Esp32
A digital clock implementation using ESP32 microcontroller that displays time on an LCD or LED display. This project demonstrates real-time clock functionality with Wi-Fi connectivity for automatic time synchronization.

![Digital Clock](./images/esp32_pinout.webp)

## Features
- Real-time clock display
- Wi-Fi connectivity for NTP time sync
- Customizable display format
- Low power consumption
- Easy setup and configuration
